# Автотесты для проверки сайта itcareerhub.de

Набор автотестов на **Python + Pytest + Selenium**, которые проверяют отображение элементов на странице https://itcareerhub.de/ru и работу кнопки с телефоном.

## Функциональность

Тесты выполняют следующие проверки:

- Отображение логотипа ITCareerHub  
- Наличие ссылок:
  - «Программы»
  - «Способы оплаты»
  - «Новости»
  - «О нас»
  - «Отзывы»
- Отображение языковых переключателей (RU и DE)
- Клик по иконке телефона и проверка текста:
  - _«Если вы не дозвонились, заполните форму на сайте. Мы свяжемся с вами»_

Все тесты запускаются через Pytest и используют браузер Chrome.
